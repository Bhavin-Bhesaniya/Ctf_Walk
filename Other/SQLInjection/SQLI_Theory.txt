SQL Injection :-
 - Malicious code input via WebPage input to modify database completly
 - Detect by given invalid parameter
 

Types of SQLi :-
 In-band SQLI :-
  Error-Based :- Error message thrown by database server to obtain information about database strcuture | Change in Parameter | ' | " | \ | OR 1=0
  Union-Based :- Base on UNION SQL Operator to combine result of two or more SELECT Statement                                 | ' UNION SELECT 1,2,3--++

 Inferential SQLi :-
  Blind SQLi :-
   - No data actually transferred via web application and attacker not able to see result of attack
   - If it execute result shown exist otherwise not exist(True/False)
   Boolean-based :- AND 1=1   | AND a=a        | ' AND 1=1 
   
   Time-based :-  
    - Given time in which if injection responding otherwise not response
    - Sleep(5)# | 1 or sleep(5)# | " or sleep(5)# | ' or sleep(5)#

 
 Out-of-band SQLi :- It depends on function being enable on database server | Attack on other plugin
